## Install Cepi/Sabre

Shell (Mac, Linux)
```shell
curl -fsSL https://raw.githubusercontent.com/Project-Cepi/localhost/main/install.sh | sh
```

Powershell (Windows)
```powershell
iwr "https://raw.githubusercontent.com/Project-Cepi/localhost/main/install.ps1" | iex
```
